# リポジトリ概要

## リポジトリ名

aws-private-lab

## 概要

AWS 系技術の技術検証用リポジトリ。
AWS の各種サービスを実際に構築・動作確認し、知見を蓄積する。

## 用途

- AWS サービスの動作検証・PoC
- IaC（AWS CDK）のパターン検証
- Lambda + Python によるアプリケーション・バッチ処理の実装検証
- アーキテクチャパターンの試行

## 方針

- 検証ごとにディレクトリを分けて管理する
- 検証結果は `docs/` 配下にメモを残す
- 使い捨てではなく、再利用可能な形で実装する
- セキュリティベストプラクティスは検証段階でも遵守する
